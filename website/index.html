About This Project

This project demonstrates Linux Server Administration on an AWS EC2 Ubuntu instance. It includes Linux user management, Apache web hosting, firewall configuration, automation using Bash scripts, scheduled backups using Cron Jobs, and system monitoring. The project showcases essential skills required for Linux System Administration, Cloud Computing, and DevOps roles.
